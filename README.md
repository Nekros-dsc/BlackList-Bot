# <a id="config"></a>⚙ » Config

If you want to change the config, open up [config.json](https://discord.gg/zM6ZN9UfRs) and locate it at the top. There you can configure the following:

*Put Token line 116 in index.js*
```js
ME_MONGO = MONGO URL
PORT=5000
```
